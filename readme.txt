#listing down the steps to deploy the above deliverables.

Clone the Git repository: First, you need to clone the repository from GitHub to your local machine.

Install Docker: If you haven't already installed Docker on your machine, download and install it from the official Docker website.

Build the Docker images: In the project root directory, create a Dockerfile that describes your application and builds the necessary Docker images.

Run the Docker container: Once the Docker image is built, you can start a container based on that image.

Access the application: Open a web browser and navigate to http://localhost:5000 to access the application.
